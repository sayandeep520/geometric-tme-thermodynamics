# geometric-tme-thermodynamics
Quantifying tumor stress via Discrete Ricci Curvature on spatial transcriptomics. We show Negative Curvature ($\kappa&lt;0$) predicts proteotoxic/hypoxic stress (HSP90, VEGFA), while Positive Curvature ($\kappa>0$) identifies fibrotic rigidity (COL1A1). Includes Colab notebooks for 10x Visium Breast Cancer &amp; Glioblastoma analysis.
